Technical Proficiencies

Github

This aim of this project was to deliver an airport check-in desk UI with certain acceptance criteria set by our trainer. To achieve this, Ben worked within an agile environment, in a team of four, experiencing a multitude of roles including scrum master, testing and development. The task was achieved through python where Ben and his team implemented OOP, TDD and SQL in order to deliver a UI that can add passengers, employees, planes and trips to their respective databases all through the UI in the python terminal.

Python

This task involved deploying a web scraper app using a CI/CD pipeline. During this task, Ben was able to create a dev branch and connect it via a webhook to a CI pipeline in Jenkins, this would run tests on the code and if successful, push the changes to the main branch. Additionally, the CD pipeline looked for any changes to the main branch and copied them to an AWS instance which acted as an ansible controller. Ansible playbooks were then run in the controller to deploy the changes to a live environment.
